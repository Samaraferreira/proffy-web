# :construction_worker: Run Web Project
```bash
# Clone Repository
$ git clone https://github.com/Samaraferreira/proffy-web.git

# Go to web folder
$ cd proffy-web

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
Go to http://localhost:3000/ to see the result.
